## Overview
This project is a machine learning based text classification system developed in Python using Jupyter Notebook.

The project includes:
- Data preprocessing and cleaning
- Feature engineering using TF-IDF
- Training multiple machine learning models
- Model evaluation and comparison

## Technologies Used
- Python
- Jupyter Notebook
- Scikit-learn
- Pandas
- NumPy
- NLTK
- Matplotlib
- Seaborn
